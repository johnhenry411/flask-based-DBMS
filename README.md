this application is a flask based DBMS that aims to provide a simple and easy to use interface for the user to interact with the database.

## Features

- Create tables
- Insert data
- Delete data
- Update data
- Select data
- Join data
- Aggregate data
- Group data
- Nested queries
- Subqueries
## Installation

```bash
$ pip install  sql-alchemy
$ pip install flask


```

## Usage

```bash

$ python3 app.py

```
